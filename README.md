# 扇贝学习工具

利用扇贝网学习英文的 chrome 扩展程序

## 下载地址

https://chrome.google.com/webstore/detail/bjnljkdheccomghaakofdlbmgcjmckgo/publish-accepted?authuser=0&hl=zh-CN

## 使用

登录扇贝网后，在任意网页(不包括扇贝网)针对英文双击查词，将单词加入扇贝生词本和学习流程
